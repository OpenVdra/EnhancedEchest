# <img src="https://skillicons.dev/icons?i=sqlite" width="28" height="28" alt="SQLite" style="display:inline-block;vertical-align:middle;margin:0 6px 0 0" /> SQLite Viewer

Open an EnhancedEchest SQLite database and browse or edit its metadata as tables, entirely in your browser. Nothing is uploaded. Your edits stay in memory while you move between documentation pages and are cleared by a full page refresh; download the database to keep them. BLOB values remain read-only to protect serialized chest data.
