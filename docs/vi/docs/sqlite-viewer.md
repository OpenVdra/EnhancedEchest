# <img src="https://skillicons.dev/icons?i=sqlite" width="28" height="28" alt="SQLite" style="display:inline-block;vertical-align:middle;margin:0 6px 0 0" /> Xem SQLite

Mở cơ sở dữ liệu SQLite của EnhancedEchest để xem hoặc chỉnh sửa metadata dưới dạng bảng, hoàn toàn trong trình duyệt. Không có gì bị tải lên. Thay đổi được giữ trong bộ nhớ khi bạn chuyển giữa các trang tài liệu và sẽ bị xóa khi refresh toàn trang; hãy tải cơ sở dữ liệu xuống để lưu lại. Các giá trị BLOB vẫn chỉ đọc để bảo vệ dữ liệu rương đã serialize.
